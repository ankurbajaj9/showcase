# showcase

Personal resume and portfolio for **Ankur Bajaj**, hosted on GitHub Pages.

🌐 **Live site:** [ankurbajaj9.github.io/showcase](https://ankurbajaj9.github.io/showcase)

## About

This repository contains the source code for the GitHub Pages resume site.  
It is built with plain HTML and CSS — no build step required.

## Structure

| File | Description |
|------|-------------|
| `index.html` | Resume page (entry point for GitHub Pages) |
| `style.css` | All styling |

## Deployment

GitHub Pages automatically publishes the `main` branch.  
Any push to `main` will update the live site within a few minutes.
